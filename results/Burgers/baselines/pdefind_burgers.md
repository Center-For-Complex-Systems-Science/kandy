# PDEFind Baseline — Inviscid Burgers

## Setup
- Domain: [0, 2π], N=128 cells, dx=0.0491
- Time: dt=0.01891, T=600 steps
- Initial condition: random Fourier modes (4 modes, seed=42)
- Data generated by: `kandy.numerics.solve_burgers` (Roe scheme, van Leer limiter)

## Method
PDEFind (Rudy et al. 2017) via pysindy.WeakPDELibrary
- Function library: PolynomialLibrary(degree=2)
- Derivative order: 2
- Optimizer: STLSQ(threshold=0.05)

## Results
| Metric | Value |
|---|---|
| Fit time (s) | 0.56 |
| 1-step MSE | 2.9634e+00 |
| R² | -167.0756 |
| Equation correct | False |

## True equation
u_t = -u * u_x   (inviscid Burgers)

## Notes
A good discovery should identify the -u*u_x term with coefficient ≈ -1.0.
The R² measures how well the discovered PDE explains time-derivative variance on the test set.
